# DeFi Lending App

A decentralized web application for interacting with the DeFi Lending smart contract.

This DApp allows users to connect their wallet and interact with the lending protocol.

## Features

- MetaMask wallet connection
- Borrow tokens from the lending contract
- Repay loan functionality
- View transaction activity
- Smart contract interaction using ethers.js

## Tech Stack

- React
- ethers.js
- Tailwind CSS
- Web3 Wallet Integration

## How it Works

1. User connects MetaMask wallet
2. DApp interacts with the smart contract
3. Users can borrow or repay tokens
4. Events are logged on the blockchain
